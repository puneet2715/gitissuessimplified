# gitissuessimplified
 Shows the current open issues on the react public repo 
 
https://user-images.githubusercontent.com/55136802/236774479-d3687901-d632-4563-915d-faf1c66b3bf9.mp4

 - clone the repo
 - cd into the repo folder ( gitissuesimplified ) 
 - run ``` npm i ```
 - run ``` npm run build ```
 - run ``` npm run preview ```
